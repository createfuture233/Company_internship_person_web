# 作品素材：个人星球 AI 助手

## 作品名称

个人星球 AI 助手

## 作品定位

这是个人网站「个人星球」中的智能辅助模块，用于帮助管理员管理内容，也帮助访客更好地理解网站中的文章和作品。

## 技术栈

- Astro
- React Islands
- NestJS
- Prisma
- SQLite
- DeepSeek API

## 核心功能

### 管理员能力

1. 后台 AI 问答  
   管理员可以询问网站内容规划、文章优化、作品分析、后台运营建议等问题。

2. 文件上传解析  
   管理员可以上传 `.txt`、`.md` 等文本类文件，系统解析文本后保存到数据库，作为 AI 生成内容的素材。

3. 生成文章或作品  
   AI 可以根据提示词、上传文件、已有文章或作品生成新的文章/作品草稿。

4. 内容分析  
   AI 可以分析已有文章或作品，从结构、表达、信息完整度、SEO、访客理解成本等角度提出建议。

5. 保存生成结果  
   管理员可以把 AI 生成的结果保存为草稿，也可以选择直接发布。

### 访客能力

1. 全站右下角有 AI 悬浮按钮。
2. 访客可以询问网站有哪些作品、某篇文章讲了什么、某个作品使用了哪些技术。
3. 访客 AI 只读取公开内容，不读取后台数据、草稿、联系信息和管理员配置。

## 设计亮点

- 管理员和访客使用不同的权限边界。
- DeepSeek API Key 只保存在后端 `.env` 中，不暴露给前端。
- AI 生成内容不会默认直接发布，需要管理员确认。
- AI 会话、消息、上传文件、生成记录都会写入 SQLite 数据库。
- 页面风格适配网站已有的明暗主题。
